✅ 4️⃣ Why MongoDB Uses BSON Instead of JSON?

Good question for class 👇

❌ Date type illa (string ah save pannum)

❌ Binary data store panna mudiyathu

BSON advantages:

Binary data support

Faster parsing

Efficient storage

Example:

  "name": "Arun",
  "createdAt": ISODate("2026-02-19")
}

BSON la date separate data type.

✅ 5️⃣ JSON vs BSON Difference

Speed Normal Faster

Used By APIs MongoDB internal

English Sentence:
JSON is text-based and human-readable, while BSON is binary and optimized for performance.
