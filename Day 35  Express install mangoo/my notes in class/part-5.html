PART 5 – Mongoose ODM Fundamentals

✅ 1️⃣ What is ORM?

ORM full form:

Object Relational Mapping

👉 SQL database use panna ORM use pannuvanga.

👉 Table ah object ah map pannum.

Example:

SQL Table → JavaScript Object

SELECT * FROM users;

ORM use pannina:

User.find()

👉 SQL query write panna thevai illa.

English Sentence:
ORM maps database tables to programming language objects.

✅ 2️⃣ What is ODM?

ODM full form:

Object Document Mapper

👉 MongoDB SQL illa.

👉 MongoDB document-based database.

👉 So ORM illa → ODM use pannuvom.

ODM map pannum:

Collection → Model

Document → Object

English Sentence:
ODM maps MongoDB documents to JavaScript objects.

✅ 3️⃣ What is Mongoose?

👉 Mongoose na Node.js ku MongoDB kuda connect panna use pannura library.

👉 MongoDB oda pesurathu easy aakum.

👉 Schema define panna allow pannum.

👉 Validation support pannum.

Install panna:

npm i mongoose

English Sentence:
Mongoose is an ODM library used in Node.js to interact with MongoDB easily.

Mongoose use pannina:

✅ 5️⃣ What is Schema?

👉 Schema na structure define pannura blueprint.

import mongoose from "mongoose" const userSchema = new mongoose.Schema({ name: String, age: Number, email: String })

👉 Inga name, age, email fields define pannom.

Schema = Rules + Structure

English Sentence:
A schema defines the structure and rules of documents in a collection.

✅ 6️⃣ What is Model?

Schema create pannina apram:

const User = mongoose.model("User", userSchema)

👉 Ithu than Model.

Model use pannithan:

Data create

Data find

Data update

Data delete

English Sentence:
A model is a constructor compiled from a schema and is used to interact with the database.

✅ 8️⃣ Validation Example

const userSchema = new mongoose.Schema({ name: { type: String, required: true }, age: { type: Number, min: 18 } })

name compulsory
