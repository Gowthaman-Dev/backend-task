1️⃣ Middleware na enna?

First concept clear pannanum.

Middleware na:

  • 👉 Request vandha udane
  • 👉 Route ku pogum munadi
  • 👉 Nadakkura intermediate function

Simple English Sentence:
Middleware is a function that runs between the request and the response.

2️⃣ CORS enna?

CORS (middleware)

CORS = Cross-Origin Resource Sharing

🔹 Origin na enna?

Origin = Protocol + Domain + Port

Example:

Frontend:

http://localhost:5173

Backend:

http://localhost:5000

Port different → Origin different.

So browser block pannum.

🔹 Browser Security Rule

Browser default ah:

  • 👉 Different origin irundha
  • 👉 Request block pannum

Error varum:

CORS policy error

3️⃣ CORS Illama enna nadakkum?

Frontend → Backend request pannanum
But browser allow panna maatan.

Server side correct irundhalum
Browser block pannum.

4️⃣ CORS eppadi solve pannum?

import cors from 'cors' movie.use(cors())

Idhu enna panudhu?

  • 👉 Backend allow pannudhu
  • 👉 "Yes, other origin request accept pannalam" nu

🔹 When CORS Not Required?

  • Frontend & Backend same port la irundha
  • Postman use pannina (browser illa)
Important Teaching Point:
CORS MongoDB ku sambandham illa ❌
Idhu browser security ku sambandham ✔

5️⃣ express.json() enna?

movie.use(express.json())

Idhu romba mukkiyam.

🔹 Problem enna?

Frontend send pannum:

{ "movieName": "Forever You" }

Without express.json():

  • 👉 req.body undefined varum

🔹 Why?

Because:

HTTP request body default ah string format la varum.

Express atha parse panna maatan automatically.

🔹 express.json() enna panudhu?

  • 👉 JSON body ah parse pannudhu
  • 👉 JavaScript object ah convert pannudhu
  • 👉 req.body la store pannudhu

Simple English Sentence:
express.json() converts incoming JSON request data into a JavaScript object.

6️⃣ HTTP Request Internal Flow

Student ku deep ah explain pannu:

Frontend send pannum:

POST /api/movie/create Content-Type: application/json

Body:

{ "movieName": "Forever You" }

Flow:

  • Request server ku varum
  • express.json() body parse pannum
  • req.body ready aagum
  • Controller access pannum

7️⃣ Important Difference (Very Very Important)

  • ❌ MongoDB connect aaganum na cors thevai illa
  • ❌ MongoDB connect aaganum na express.json thevai illa
  • ✔ HTTP request handle panna mattum thevai

Ithu kandippa class la highlight pannu.

8️⃣ Middleware Order Importance

movie.use(cors()) movie.use(express.json()) movie.use("/api/movie", movieroute)

Why above route?

Because:

  • Middleware first run aaganum
  • Appo than route correct ah work aagum

Wrong:

movie.use("/api/movie", movieroute) movie.use(express.json())

Ithu pannina req.body undefined varum.
