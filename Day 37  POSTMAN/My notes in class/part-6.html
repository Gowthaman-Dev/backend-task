✅ Part 6 – Schema Creation (Most Important Part)

இது MongoDB + Mongooseல ரொம்ப முக்கியமான concept 🔥 Schema clearஆ புரிஞ்சா தான் DB architecture strong ஆகும்.

🔹 1️⃣ What is Schema?

👉 Schema = Structure of Document

MongoDBல data JSON formatல store ஆகும். ஆனா எந்த field வரணும்? எந்த type? default value என்ன? இதெல்லாம் define பண்ணுறது தான் Schema.

const userSchema = new mongoose.Schema({ name: String, email: String, age: Number })

இதன் அர்த்தம்:

  • name → String
  • email → String
  • age → Number

🎯 Schema = Blueprint of Collection

🔹 2️⃣ Why Schema Mandatory?

MongoDB itself schema-less database.

அப்படின்னா schema தேவையில்லையா?

❌ Direct MongoDBல தேவையில்லை ✅ But Mongoose use பண்ணும்போது mandatory

ஏன்?

  • Data validation
  • Proper structure
  • Type checking
  • Default values
  • Security

Without schema:

{ name: 123, email: true }

Wrong data save ஆகிடும் 😵

With schema: Mongoose error throw பண்ணும்.

👉 அதனால தான் schema mandatory.

🔹 3️⃣ Why Constructor? (new mongoose.Schema())

Schema ஒரு constructor function.

Constructorனா என்ன?

👉 Object create பண்ண பயன்படும் special function.

நாம் new keyword use பண்ணி instance create பண்ணுறோம்.

new mongoose.Schema()

இதுல internally:

  • New Schema object create ஆகும்
  • Fields store ஆகும்
  • Validation rules attach ஆகும்
  • Options attach ஆகும்

🔹 4️⃣ import mongoose

import mongoose from "mongoose"

Mongoose = ODM (Object Data Modeling) library.

👉 MongoDB & Node.js க்கு நடுவில் bridge மாதிரி work பண்ணும்.

Architecture: Node App → Mongoose → MongoDB

🔹 5️⃣ Field Types

🔸 String

name: String

Store text data.

Example:

"Gowtham"

🔸 Default Value

isActive: { type: Boolean, default: true }

User value குடுக்கலேன்னா Default automatically save ஆகும்.

Example saved document:

{ "name": "Gowtham", "isActive": true }

🔹 6️⃣ timestamps: true

const userSchema = new mongoose.Schema({ name: String }, { timestamps: true })

இது enable பண்ணா automatically இரண்டு fields add ஆகும்:

  • createdAt
  • updatedAt

🔹 createdAt

Document first time save ஆகும் போது automatic date store ஆகும்.

Example:

"createdAt": "2026-02-20T05:00:00.000Z"

🔹 updatedAt

Document update பண்ணும் போது automatic update time change ஆகும்.

🔥 Internally என்ன நடக்குது?

  • Schema create ஆகுது
  • Mongoose model உருவாக்குது
  • Model MongoDB collection connect ஆகுது
  • Save பண்ணும் போது:
  • Validation check
  • Default values set
  • Timestamps add
  • Then MongoDBக்கு send

🎯 Full Example

import mongoose from "mongoose" const userSchema = new mongoose.Schema({ name: { type: String, required: true }, email: String, isActive: { type: Boolean, default: true } }, { timestamps: true }) const User = mongoose.model("User", userSchema) export default User

🧠 DB Architecture Clear View

Schema → Model → Collection → Document Schema = Structure Model = Controller Collection = Table Document = Row

🚀 Simple Real-Time Flow

Client → Route → Controller → Model → Schema → MongoDB
