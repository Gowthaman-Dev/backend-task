1️⃣ Project Create Pannuradhu

Step 1 – Folder Create

Example:

movie-backend

Inside open terminal:

npm init -y

👉 Idhu package.json create pannum.

2️⃣ package.json Explain (Class la kandippa sollanum)

Important fields:

{ "name": "movie-project", "version": "1.0.0", "main": "server.js", "type": "module", "scripts": { "start": "node server.js", "dev": "nodemon server.js" } }

🔹 main

Entry file (server.js)

🔹 type: module

Why important?

ES6 import/export use panna.

Without this:

import express from 'express'

work aagathu.

Simple English Sentence:
"type: module" allows us to use ES6 import and export syntax in Node.js.

3️⃣ Required Packages Install

npm install express mongoose cors dotenv npm install nodemon --save-dev

🔹 express

Express.js

👉 Backend server create panna use pannuvom.

🔹 mongoose

Mongoose

👉 MongoDB connect panna use pannuvom.

🔹 cors

Cross Origin Resource Sharing handle panna.

Frontend different port la irundha thevai.

🔹 dotenv

Environment variables manage panna.

Example:

PORT=5000 MONGOOSE_URL=mongodb://localhost:27017/movie

🔹 nodemon

Server automatic restart panna use pannuvom.

Without nodemon:
Every change ku manually restart pannanum.

4️⃣ Folder Structure (Most Important Teaching Point)

Clean architecture romba mukkiyam.

movie-backend/ │ ├── config/ │ └── db.js │ ├── modules/ │ └── movieDataModule.js │ ├── controller/ │ └── movieController.js │ ├── routes/ │ └── movieRoute.js │ ├── .env ├── server.js └── package.json

5️⃣ Why Separate Folders? (Class la strong ah sollu)

🔹 config

Database connection files

🔹 modules

Schema + Model files

🔹 controller

Business logic

🔹 routes

API endpoint definitions

🔹 server.js

Main entry file

6️⃣ Architecture Understanding (Super Important)

Frontend → Route → Controller → Model → MongoDB

Ithu backend flow.

7️⃣ Why Not Everything in One File?

  • If everything in one file:
  • 1000+ lines
  • Debug panna kashtam
  • Team work impossible
  • Maintain panna mudiyathu

So separation of concerns follow pannuvom.

Simple English Sentence:
We separate folders to keep the backend clean, scalable, and maintainable.

8️⃣ Environment File (.env)

Create:

PORT=5000 MONGOOSE_URL=mongodb://localhost:27017/movie

⚠ Important:
.env GitHub ku push panna koodathu.

So create .gitignore:

node_modules .env
