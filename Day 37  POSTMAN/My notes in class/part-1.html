1️⃣ Database na enna?

Example:

  • Movie details
  • User details
  • Orders
  • Products

Database is a structured place where we store and manage data.

2️⃣ MongoDB na enna?

MongoDB create pannina database system dhaan MongoDB.

MongoDB is a NoSQL Database.

👉 SQL madhiri table-based illa.
👉 JSON format la data store pannum.

6️⃣ ORM vs ODM (Super Important)

🔹 ORM – Object Relational Mapping

ORM SQL database kku use pannuvom.

Example:

  • MySQL
  • PostgreSQL

ORM la:
Table ↔ Class mapping

🔹 ODM – Object Document Mapping

MongoDB ku use pannuvom.

Document ↔ Object mapping

7️⃣ Mongoose enna?

Mongoose is a ODM library for MongoDB.

👉 It helps Node.js connect with MongoDB
👉 It gives structure
👉 It gives validation
👉 It gives built-in methods

Simple English Sentence:
Mongoose is an ODM library that connects Node.js applications with MongoDB.

8️⃣ Why Mongoose Important?

Without mongoose:

  • Direct MongoDB driver use pannanum
  • Manual validation
  • No schema control

With mongoose:

  • Schema define pannalam
  • Validation automatic
  • Default values set pannalam
  • Timestamps automatic

9️⃣ Why Schema Important?

MongoDB flexible database.

But real project la structure illama irundha problem varum.

Example:

User1:

{ "name": "A" }

User2:

{ "username": "B" }

Structure mismatch varum.

So schema use pannuvom to control structure.
