3️⃣ server.js Full Explanation

import express from 'express'
import dotenv from 'dotenv'
import cors from 'cors'
import userRoute from './routes/userRoute.js'
import ConnectDb from './config/db.js'

dotenv.config()

ConnectDb()

const app = express()

app.use(cors())
app.use(express.json())

app.use("/api/user", userRoute)

const PORT = process.env.PORT || 3000

app.listen(PORT,()=>{
    console.log(`Server running http://localhost:${PORT}`);
})

Line by Line Explanation

🔹 dotenv.config()

👉 .env file variables load ஆகும்

🔹 ConnectDb()

👉 Database connection start ஆகும்

🔹 const app = express()

👉 Server create ஆகும்

🔹 app.use(cors())

👉 Cross origin allow செய்யும்

🔹 app.use(express.json())

👉 JSON data read செய்ய உதவும்

(Without this req.body work ஆகாது ❌)

🔹 app.use("/api/user", userRoute)

👉 Route connect ஆகும்

Meaning:

http://localhost:3000/api/user

4️⃣ Route File Explanation

routes/userRoute.js

import express from "express";
import { addData, getData } from "../controllers/userController.js";

const router = express.Router();

router.post("/add", addData);
router.get("/get", getData);

export default router;

Explanation

🔹 express.Router()

👉 Mini router create ஆகும்

🔹 router.post("/add", addData)

Meaning:

POST /api/user/add

Flow:

Client → Route → Controller

🔹 router.get("/get", getData)

Meaning:

GET /api/user/get

5️⃣ Controller Deep Explanation

controllers/userController.js

import userModel from "../models/userModel.js";

export const addData = async(req,res)=>{
 try {

  const { userName, userEmail, userAge } = req.body

  const add = await userModel.create({
    userName,
    userEmail,
    userAge
  })

  res.status(201).json({msg:"successfully added"})

 } catch (error) {
  res.status(500).json({msg:"error", error})
 }
}

7️⃣ How Request Travel?

Example:

POST

http://localhost:3000/api/user/add

{
 "userName":"Gowtham",
 "userEmail":"gow@gmail.com",
 "userAge":22
}

Travel:

  • Route match ஆகும்
  • Controller run ஆகும்
  • Model.create() execute ஆகும்
  • MongoDB store ஆகும்
  • Response send ஆகும்
