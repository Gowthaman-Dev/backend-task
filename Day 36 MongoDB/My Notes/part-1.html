5️⃣ What is Schema?

👉 Schema என்றால் என்ன?

Schema = Blueprint of Collection

Schema சொல்வது:

  • ✔ என்ன field allowed
  • ✔ என்ன type data
  • ✔ validation என்ன
  • ✔ required or not
  • ✔ default value

Your Schema Example:

const userSchema = new mongoose.Schema({ userName:{type:String,required:true}, userEmail:{type:String,required:true,unique:true}, userAge:{type:Number,required:true,min:18} },{timestamps:true})
FieldMeaning
type:Stringstring only
required:truecompulsory
unique:trueduplicate not allowed
min:18age must be 18+
timestamps:truecreatedAt, updatedAt auto add

6️⃣ What is Model?

👉 Model என்றால் என்ன?

Model என்பது:

Schema ல இருந்து create ஆகும்.

const userModel = mongoose.model('userDatas',userSchema)

👉 userDatas = collection name
👉 userSchema = structure

Model Meaning:

  • 🔹 Database collection க்கு reference
  • 🔹 CRUD operations செய்ய use ஆகும்
  • 🔹 ORM methods model குள்ள தான் இருக்கும்

👉 "Model is a wrapper around MongoDB collection"

9️⃣ Why Schema Important?

Without schema:

  • ❌ Random data store ஆகும்
  • ❌ Type mismatch வரும்
  • ❌ Data consistency போயிடும்

Schema use பண்ணினால்:

  • ✔ Validation
  • ✔ Structure
  • ✔ Clean data
